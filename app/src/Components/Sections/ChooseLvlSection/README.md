# Choose Level Section
## Props
### Levels prop
**Format**: 
```javascript
levels = {[
  {
    name: "Level 1",
    description: "This is level for begginers.",
    value: 1
  },
  {
    name: "Level 2",
    description: "This is level for advanced players.",
    value: 2
  },
  {
    name: "Level 3",
    description: "This is level for masters.",
    value: 3
  }
]}
```
